# Ruquiya-mern-10pshine
**Cognify**
Cognify is a modern full-stack notes management system that allows users to create, edit, organize, and manage notes efficiently. It’s designed with scalability, clean UI, and robust testing in mind — making note management simple, secure, and reliable.
**Features:**
Create, update, and delete notes easily
Create, update, and delete Folder easily
Categorize and organize notes by type, tag
Secure data storage using MongoDB
Intuitive and responsive dashboard UI
Comprehensive testing with Jest and Mocha/Chai
Code quality analysis using SonarQube
Application logging powered by Pino Logger

**Tech Stack**
Frontend: React.js
Backend: Node.js, Express.js
Database: MongoDB
Testing: Jest, Mocha, Chai
Tools: SonarQube, Pino Logger

**Installation & Setup**
Clone the repository
git clone https://github.com/Ruquiya/Ruquiya-mern-10pshine.git
cd Ruquiya-mern-10pshine

Install dependencies
npm install
cd backend && npm install

Set up environment variables
Create a .env file in the backend directory and add:
MONGO_URI=your_mongodb_connection_string
PORT=5000

Run the application
npm run dev

Test
Run npm test in backend and frontend folder
